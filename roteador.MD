# CONFIGURAÇÃO ROTEADOR

## CONEXÃO LOCAL
### GATEWAY
```
EN
CONF T
HOST RT-BH
INT F0/0
NO SH
IP AD 192.168.0.1 255.255.255.0
DESC LAN BH-GATEWAY
EXIT
EXIT
SH R
COPY R ST
```

### DHCP
```
EN
CONF T
INT F0/0
IP DHCP EXCLUDED-ADDRESS 192.168.0.1
IP DHCP POOL LAN-BH
NET 192.168.0.0 255.255.255.0
DEFAULT-ROUTER 192.168.0.1
EXIT
EXIT
SH R
COPY R ST
```
